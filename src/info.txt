
Stannum

Stannum is a work-in-progress kernel with FAT12 support, and soon, maybe even
file writing routines, runtime services, drivers, and all that fun stuff that
makes usability

Done:
* Simple command line
* Proper block allocation for programs and files

Priority:
* Better memory allocation system for programs instead of 6 fixed 64KiB chunks
    * Terminate and stay resident
* Subdirectories
* File writing
    * ed text editor
